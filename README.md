Hello, I am Omar and these are my dotfiles for hyprland. I am very new to this but I wished to upload these so that I can always access them and share them.
Run this to install, it will delete the "dotfiles" folder if it is there and make a new one, make sure to rename your dotfiles folder if you wish to save your dot files

rm -rf ~/dotfiles && git clone https://github.com/1616856/dotfiles.git ~/dotfiles && bash ~/dotfiles/install.sh
